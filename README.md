# SEO Content Architect

SEO Content Architect is a Windows app for generating SEO‑optimized articles and images with AI. It supports WordPress auto‑posting, advanced scheduling, and multiple AI models. The free version includes limited features and a 15‑day trial for automated publishing.

## Overview

SEO Content Architect is a Windows desktop application built with Electron, React, and Vite.  
It automates the entire content workflow — from AI‑powered article generation to WordPress publishing — while maintaining full local control and offline reliability.

## Key Features

- AI‑generated, SEO‑optimized articles and images  
- Integrated content editor  
- Smart content source management  
- Automated WordPress posting and scheduling  
- Local backend for cron jobs, backups, and AI services  
- Windows build support via Electron Builder
